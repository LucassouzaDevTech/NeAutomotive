# NE Automotive

Site institucional desenvolvido para a **NE Automotive**, oficina de peças e serviços automotivos em São Jerônimo da Serra, PR.

## Tecnologias

- HTML5 · CSS3 · JavaScript vanilla
- Google Fonts — Bebas Neue, Barlow Condensed, Barlow
- Google Maps Embed

## Funcionalidades

- Design responsivo para mobile e desktop
- Animações de scroll com Intersection Observer
- Navegação com menu mobile
- Galeria de serviços
- Integração direta com WhatsApp
